# 树莓派磁盘清理脚本 - 设计文档

## 概述

为树莓派家庭服务器设计一个通用的磁盘空间清理脚本，支持交互式菜单和命令行参数两种使用方式。

## 需求背景

- 目标用户：树莓派家庭服务器用户
- 存储空间：128GB+（空间宽裕）
- 使用场景：通用清理 + 可选模块（Docker、npm/yarn/pnpm、pip）

## 设计方案

### 使用方式

#### 交互式菜单（默认）

支持多选清理模块：

```
===========================
  树莓派磁盘清理工具
===========================
[ ] 1. APT 缓存清理
[*] 2. 日志文件清理
[ ] 3. 临时文件清理
[*] 4. 缩略图缓存清理
[ ] 5. Docker 清理 (可选模块)
[ ] 6. npm/yarn/pnpm 清理 (可选模块)
[ ] 7. pip 缓存清理 (可选模块)
[ ] 8. 显示磁盘使用情况

已选择: 2, 4
请选择 [1-8, 空格切换选择, 回车确认, 0 退出]:
```

操作说明：
- `空格键`：切换选择
- `回车键`：确认执行
- `0`：退出

#### 命令行参数

```bash
# 交互式菜单（默认）
./cleanup.sh

# 命令行参数
./cleanup.sh --all              # 全部清理
./cleanup.sh --apt --logs       # 清理 apt 和日志
./cleanup.sh --docker --pip     # 清理 docker 和 pip
./cleanup.sh --dry-run          # 预览模式
./cleanup.sh --help             # 显示帮助
```

### 清理模块

| 编号 | 模块 | 清理内容 |
|------|------|----------|
| 1 | APT 缓存 | `/var/cache/apt/archives/` |
| 2 | 日志文件 | `/var/log/` 下的日志文件（保留最近7天） |
| 3 | 临时文件 | `/tmp`、`/var/tmp` |
| 4 | 缩略图缓存 | `~/.cache/thumbnails/` |
| 5 | Docker (可选) | dangling 镜像、停止的容器、未使用的网络 |
| 6 | npm/yarn/pnpm (可选) | npm cache、yarn cache、pnpm store |
| 7 | pip (可选) | pip cache (`~/.cache/pip`) |

### 预览模式

清理前显示：
- 每个模块将清理的内容列表
- 每个模块预估可释放的空间大小
- 总计可释放空间

### 安全确认

执行清理前显示汇总：
```
===========================
  清理确认
===========================
已选择模块:
  - 日志文件清理
  - 缩略图缓存清理

预计释放空间: 256 MB

确认执行清理? [y/N]:
```

### 错误处理

- 权限不足时提示使用 sudo
- 某模块清理失败时继续执行其他模块
- 完成后显示清理结果摘要

## 技术实现

- 脚本语言： Bash
- 依赖工具： du、df、find、apt、docker（可选）
- 兼容性： Raspberry Pi OS、Ubuntu 等 Debian 系系统
